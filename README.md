minimal-flux
=========

A very lightweight implementation of Flux

[![Build Status](https://travis-ci.org/malte-wessel/minimal-flux.svg?branch=master)](https://travis-ci.org/malte-wessel/minimal-flux)

## Install

````
npm install minimal-flux --save
````
